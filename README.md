# Airline Flights Data Analysis  

## Overview  
Performed exploratory data analysis (EDA) on airline flight data to uncover patterns in pricing, duration, and airline performance using Python.

---

## Key Highlights  
- Conducted end-to-end EDA on real-world flight dataset  
- Cleaned and preprocessed data (handled missing values, feature formatting)  
- Analyzed relationships between price, duration, stops, and airlines  
- Created visualizations to identify trends and correlations  

---

## Tech Stack  
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook  

---

## Key Insights  
- Flight prices vary significantly with **number of stops and duration**  
- Certain airlines show **consistent pricing patterns across routes**  
- Identified key factors influencing ticket pricing
